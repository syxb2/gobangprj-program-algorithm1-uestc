这是一个基于 c 语言和 easyx 图形库的五子棋游戏，支持双人对战和人机对战。

### Environment

操作系统：`windows`

easyx 图形库可以从 easyx 官网`https://easyx.cn`下载并安装在`Visual Studio`中

### Usage

* 使用`Visual Studio 2022`打开文件夹中的`.sln`文件即可打开此解决方案
* 打开`.exe`文件使需要`pictures`文件夹和`.exe`文件位于同一目录下
